# AI 면접관 Agent v2.0: 전략적 질문 생성 및 피드백 시스템

## 프로젝트 개요

이 프로젝트는 **LangGraph**를 기반으로 지원자의 **이력서를 분석**하고, 맞춤형 **면접 전략**에 따라 **심층 질문을 생성**하며, 답변에 대한 **정량/정성 피드백을 제공**하는 AI 면접관 Agent 시스템입니다. 지원자에게 실전 면접 감각과 체계적인 자기 분석 기회를 제공하는 것을 목표로 합니다.

### 핵심 목표 및 문제 해결

| 구분 | 내용 |
| :--- | :--- |
| **문제 정의** | 채용 과정의 높은 부담, 면접 후 **개인화된 피드백 부재** (무엇이 부족한지 알 수 없음). |
| **프로젝트 목표** | 이력서 기반의 **개인화된 질문**과 **객관적인 평가 및 피드백**을 제공하는 Agent 구축. |
| **핵심 기술** | `LangGraph` 기반의 **전략적 질의응답 흐름 제어** (Flow Control) 구현. |

## 핵심 기여 파트: 면접 진행 검토 고도화 (Decide Next Step)

저는 Agent의 **유연한 면접 진행 흐름**을 담당했으며, LLM의 상황 판단 능력과 규칙 기반 로직을 결합하여 사람처럼 자연스러운 심층 인터뷰를 구현했습니다.

### 1. **핵심 기능: 전략적 흐름 제어**

Agent는 단순히 정해진 횟수만큼 질문하는 것이 아니라, **지원자의 답변 품질**과 **남은 면접 주제**를 고려하여 다음 행동(심화 질문, 주제 전환, 종료)을 결정합니다.

| 노드 | 역할 및 구현 내용 | 기여 내용 |
| :--- | :--- | :--- |
| **`decide_next_step`** | **LLM 자율 판단 로직 구현:** 단순 `if-else` 대신 LLM에게 최종 평가와 남은 주제를 제공하여 **'generate' (심화), 'next\_topic' (주제 전환), 'end' (종료)**를 자율적으로 결정하게 함. | **사람다운 면접 구현:** 답변의 질이 낮으면 주제를 심화하고, 답변이 좋으면 다음 주제로 넘어가는 유연한 면접 흐름 확보. |
| **`next_topic_question`** | **질문 큐(Queue) 시스템 구현:** 전략 딕셔너리에서 다음 주제의 첫 질문을 꺼내어 `current_question`에 할당하고, 사용된 주제를 `remaining_topics`에서 제거하여 **준비된 질문을 안정적으로 소진**하는 로직 구축. | **안정성/효율성 확보:** 면접이 무한 루프에 빠지거나, 정해진 주제를 빠짐없이 질문하는 체계적인 구조 완성. |

### 2. **평가 체계 고도화**

*   **다차원 평가 기준:** 기존 연관성/구체성 외에 **영향/성과, 역할·주도성, 커뮤니케이션** 등 5개 항목을 추가하여 **입체적인 평가**를 수행.
*   **가중합 시스템:** 각 평가 항목에 가중치(예: 연관성 30%)를 부여하여 객관적인 **'가중합 점수'**를 도출하고, 이를 합격 가능성 판단의 핵심 지표로 활용.

## 기대 효과

1.  **실전 감각 향상:** 이력서 기반의 날카로운 심층 질문을 반복 훈련하여 실전 면접 준비를 극대화.
2.  **객관적 피드백:** 5가지 기준으로 점수화된 객관적 피드백을 통해 지원자 스스로 강점과 약점을 명확히 인지.
3.  **LLM 활용 역량:** LangGraph를 통한 **복잡한 의사결정 흐름(Routing)** 설계 능력을 습득.
